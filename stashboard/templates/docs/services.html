Services API

The Services List resource represents all web services currently tracked via StashBoard. The resources also allows for the creation of new, trackable web services.

Attributes

id
The unique identifier by which to identify the service>
name
The name of the service, defined by the user
description
The description of the web service
service-list
The service list this service belongs to, if any
current-event
The current event for the service, if any
url
URL of this resource
